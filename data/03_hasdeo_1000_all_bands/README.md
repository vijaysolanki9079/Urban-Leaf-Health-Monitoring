# Hasdeo 1000 All Bands Dataset

This folder contains logs and metadata for the Hasdeo 1000 dataset. The dataset includes high-quality images in .tiff format, covering all bands.

## Contents
- **Logs**: Task IDs and logs for tracking data processing.
- **Metadata**: Export logs and manifest files summarizing the dataset.

## Purpose
The data in this folder is used for:
- Collecting and processing high-quality images for analysis.
- Preparing data for augmentation and preprocessing.

## Data Download
The dataset of 1000 images in .tiff format (all bands) is available for download at the following link:
