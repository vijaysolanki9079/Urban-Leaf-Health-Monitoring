# Comparison Based on Events

This folder contains data and resources for event-based comparisons. The data includes cleaned image features, metadata, and logs for analyzing changes over specific events.

## Contents
- **CSV Files**: Contains cleaned image features, correlation data, and export logs.
- **Logs**: Task IDs and other logs for tracking data processing.
- **Metadata**: Export logs and manifest files summarizing the dataset.
- **Tiff to JPEG**: Converted images for analysis.

## Purpose
The data in this folder is used for:
- Comparing changes in the area based on specific events (e.g., deforestation, urbanization).
- Visualizing features using GEE APIs.
