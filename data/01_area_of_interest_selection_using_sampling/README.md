# Area of Interest Selection Using Sampling

This folder contains sampled data batches for different regions. The data is organized into batches to facilitate targeted analysis and processing.

## Contents
- **Batch 1**: Data for the Hasdeo Forest dataset (RGB images and metadata).
- **Batch 2**: Data for the Sydney Blue Mountains fringe.
- **Batch 3**: Data for the Kangaroo Island Black Summer.

## Purpose
The data in this folder is used for:
- Sampling and selecting areas of interest for analysis.
- Preparing data for further processing and augmentation.
